## 第三种：java -jar jar包 命令启动

### 1：首先在idea下给项目打包

![image-20201029122145003](https://cdn.phpmianshi.cn/PicGo/image-20201029122145003.png)

### 2：在jar包所在目录打开命令行，然后输入命令启动

```shell
java -jar XXXXXX.jar
```
