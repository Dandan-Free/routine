# php打印格式化显示利器<pre>

当我们调试php代码的时候，会使用var_dump、print_r 输出结果，输出的时候需要格式化，这就需要<pre>

### 格式化后的结果

```php
echo "<pre>";
$arr=[3,1,5,8];
print_r($arr);
```
